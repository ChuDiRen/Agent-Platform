# 测试数据生成技能

## 数据类型

### 正常数据
- 符合业务规则的有效数据
- 各字段值在合法范围内
- 关联数据一致 (如 user_id 对应存在的用户)

### 边界数据
- 字段最小值/最大值
- 空字符串 vs null
- 最短/最长输入

### 异常数据
- 类型错误 (字符串填数字字段)
- 超长数据
- SQL注入 / XSS payload
- 特殊字符 (emoji, 全角, 换行)

## 字段生成规则

| 类型 | 正常值 | 边界值 | 异常值 |
|------|--------|--------|--------|
| email | test@example.com | a@b.c | 无@符号 |
| phone | 13812345678 | 13000000000 | 123abc |
| password | Test@1234 | aB1@ | 123 |
| age | 25 | 0, 150 | -1, abc |
| date | 2026-01-01 | 1970-01-01 | 2026-13-01 |

## 输出格式
- **JSON**: 结构化数据，适合 API 测试
- **CSV**: 表格数据，适合批量导入
- **SQL**: INSERT 语句，适合数据库测试
- **Faker**: 使用 faker 库生成逼真数据
